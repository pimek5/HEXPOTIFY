# Changelog - DynamicGlow Theme

## [1.0.0] - 2024-12-08

### Pierwsza wersja!

#### Dodane funkcje:
- ✨ **Dynamiczne kolory** - automatyczne dostosowanie do okładki albumu
- 💫 **Efekty glow** - świetlne efekty wokół wszystkich elementów UI
- 🌓 **Tryb Light/Dark** - przełącznik w prawym górnym rogu
- 🖼️ **Animowane tło** - płynnie zmieniające się tło z okładką
- 🎵 **Skupione teksty** - dynamiczne zoom na aktywny tekst piosenki
- ⚡ **Płynne przejścia** - wszystkie animacje są smooth
- 🎨 **4 schematy kolorów** - Dark/Light Base i Animated

#### Pliki:
- `color.ini` - Definicje schematów kolorów
- `user.css` - Połączone style z DefaultDynamic + Glowify
- `dynamic-glow.js` - Główny skrypt łączący funkcjonalności
- `Vibrant.min.js` - Biblioteka do analizy kolorów
- `install.ps1` - Automatyczny instalator dla Windows
- Pełna dokumentacja w README.md
